# NestJS Template Project

## Installation

```bash
git clone https://github.com/r253-dev/NestJSTemplate.git
cd NestJSTemplate
$ npm ci
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
